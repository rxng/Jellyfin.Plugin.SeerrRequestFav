SeerrRequestFav

Automatically request movies and TV shows in Jellyseerr when you mark them as a favourite in Jellyfin.

Jellyseerr Connection

When disabled the plugin will not process any favourite events or run syncs.
Base URL of your Jellyseerr/Overseerr instance (no trailing slash).
Found in Jellyseerr → Settings → General.

Favourite Request Behaviour

Immediately send a request to Jellyseerr when an item is marked as favourite. When disabled you must use the Sync Favourites button manually.
Remove the favourite flag once the item has been successfully requested in Jellyseerr.
When requesting a TV series, only request season 1 instead of all seasons.
Advanced Settings
Elevates debug-level log entries to Information so they appear in the Jellyfin log.
Very verbose – logs individual API calls and item-level detail.
